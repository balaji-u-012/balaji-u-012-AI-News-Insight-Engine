# AI News Aggregator

An AI-powered personalized news digest platform that collects the latest AI news, research papers, blogs, and videos, ranks them using LLMs, and delivers daily email digests to users.

---

# Features

- User Registration & Login
- JWT Authentication
- Personalized AI News Feed
- Daily Email Digest
- AI-based Content Ranking
- Research Paper Aggregation
- OpenAI & Anthropic Blog Scraping
- YouTube AI Video Aggregation
- PostgreSQL Database
- FastAPI Backend
- React Frontend
- APScheduler Automated Email Delivery

---

# Tech Stack

## Backend
- Python

## Frontend
